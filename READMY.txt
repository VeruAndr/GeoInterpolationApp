# Геоинтерполятор - The Interpolation pROgram Service , или вкратце "GEO TIROS"
(одноименная деревня в Греции и первый американский метеорологический спутник)
-----------------
Содержание архива:
| example_data.csv - пример входных данных для программы
| TIROS_v6.py - программа Геоинтерполятор
-----------------

Программа для интерполяции геоданных с графическим интерфейсом. Позволяет визуализировать пространственные данные и создавать карты распределения параметров.

## Возможности

### 📊 Интерполяция
- **B-сплайн** - гладкая интерполяция
- **TIN (Triangulation)** - триангуляция Делоне
- **IDW** - метод обратных взвешенных расстояний

### 🗺️ Визуализация
- Цветовые карты распределения
- Изолинии с настраиваемыми параметрами
- Подписи точек
- Фоновая карта-подложка
- Настройка прозрачности слоев

### 🎛️ Интерфейс
- Компактная левая панель с прокруткой
- Возможность скрыть панель для детального просмотра
- Интерактивная навигация по графику
- Сохранение результатов в различных форматах

### 📁 Поддержка данных
- CSV и TXT файлы
- Автоопределение разделителей
- Обработка пропущенных значений
- Возможность замены координат (широта/долгота)

## Установка

### Требования
- Python 3.8 или выше
- pip (менеджер пакетов)

### Установка зависимостей

```bash (Linux MacOS)
# Установка всех необходимых модулей
pip install pandas numpy scipy matplotlib Pillow
Или через файл
pip install -r requirements.txt


ЗАПУСК ПРОГРАММЫ:
python TIROS.py
